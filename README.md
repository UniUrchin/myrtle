## ✨ Overview

Myrtleは、日々抱えている嘆きを心ゆくまで呟けるSNS(Solo Nageki System)です。
頑張って開発中なので見守ってね。

## 🔨 Development

- 開発用のサーバを立ち上げる

  ```
  $ npm run dev
  ```

- Chakra UI用のカスタムテーマを出力する

  ```
  $ ./node_modules/.bin/chakra-cli tokens ./src/styles/theme.ts
  ```
